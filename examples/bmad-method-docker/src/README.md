## Getting Started

1. Run `bmad install` to initialize BMAD-METHOD in this project
2. Use BMAD agents to plan and implement features
3. Follow the Agentic Agile Development workflow

## Development with BMAD

BMAD-METHOD provides AI agents to assist throughout the development lifecycle:

- **Analyst**: Requirements gathering and analysis
- **PM**: Project management and planning
- **Architect**: System design and architecture
- **Scrum Master**: Story breakdown and sprint planning
- **Dev**: Implementation assistance
- **QA**: Testing and quality assurance

## Example Workflow

```bash
# Check BMAD installation
bmad status

# Flatten codebase for LLM context
bmad flatten

# Check for available expansion packs
bmad list:expansions
```

Refer to the [BMAD-METHOD documentation](https://github.com/bmad-code-org/BMAD-METHOD) for detailed guidance on using the framework.
